# Contributing to Test-Project-Cadence

Thank you for your interest in contributing! 

## How to Run Locally

This project consists of static files. You can run it locally using any static web server.

### Option 1: Using Node.js (npx)
```bash
npx serve .
```

### Option 2: Using Python
```bash
# For Python 3
python3 -m http.server 8000
```

Once the server is running, open your browser and navigate to `http://localhost:8000` (or the port specified by the server).

## Development Workflow
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Make your changes.
4. Ensure `index.html` and `health.json` are still accessible.
5. Submit a pull request.
